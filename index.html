Navigation

  • Remember those arrow keys?

    Forget them, Vim is about keeping your fingers where they need to be

    ,---,---,---,---,
|←  |   |↑↑↑|  →|
|←h | j | k | l→|
|←  |↓↓↓|   |  →|
`---`---`---`---`

Navigation - Cont

  • Repeat Yourself

    10j (move down 10), 5l (ell, move right 5)

    (You can do this with most commands!

  • Jump around

    0 (zero, go to very start of line)

    ^ (go to first non-blank character)

    $ (go to end of line)

Navigation - Advanced

  • ctrl+f (page forward, err, down)

    ctrl+b (page back, err, up)

    gg (go to beginning of file)

    G (go to end of file)

  • And many many more (jump forward a word, go to the next __ character, go to top of screen, parenthesis matching, etc.)

Now try it

Vim text1.txt, move your cursor around (don't quit!)

  1. $vim text1.txt

  2. Notes:

    • h, j, k, l for navigation
    • 0, ^, $ to jump around a line
    • try multipliers

Semi-Advanced Editing

  • Alternatives for i

    A -> input mode (at end of line)

    o -> (lower O) inserts new line (below cursor row)

  • Other fancy things

    R -> start REPLACE mode

    x -> delete character under cursor

Remember when things were case sensitive?

  • Most things have a similar function on same key

    a -> input mode (one space after cursor)

    O -> (upper O) inserts new line (at cursor row)

    r -> replace one character under cursor

    X -> delete character before cursor

  • This applies to other things too!

    (Like, maybe the navigation buttons)

Search

  • Your first foray into Command-line mode

    this isn't all you can do in command-line mode

    you will learn about those later

  • / -> then type what you want

    /fish -> search for "fish"

    use a regex: /^fish -> search for "fish" at beginning of line

  • n or N to iterate through results

Replace pay attention!

  • :%s/needle/replacement/

  • :%s/needle/replacement/g

Looks like sed?

  • :s/needle/replacement/ (is for current line)

    :5,12s/needle/replacement/ (from line 5 to 12)

    :.,$/needle/replacement/ (current line to end of file)

    :.,+2/needle/replacement/ (current line and next 2)

Replace your visually marked stuff!

  • :'<,'>s/needle/replacement/

    You don't even need to type in the '<,'> part

Notes:

  • :%s/needle/replacement/ (or add g)
